## Overview
TravelBook is a web application designed to facilitate the rental accommodation process for travelers and property owners alike. With intuitive booking management tools, advanced search and filter functionalities, and a responsive design, TravelBook aims to enhance the overall rental experience. This README provides an overview of our project, including our team members and file organization.
